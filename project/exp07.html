JavaScript String and Math Functions

String Functions

The JavaScript String object functions.

1. length

The length property returns the length of a string:

var str = "Hello, World!";
var length = str.length;
console.log(length); // Output: 13

2. charAt

The charAt() function returns the character at a specified index in a string:

var str = "Hello, World!";
var char = str.charAt(0);
console.log(char); // Output: H

Math Functions

The JavaScript Math object provides mathematical functions and constants.

1. round

The Math.round() function returns the nearest integer to a given number:

var num = 3.7;
var roundedNum = Math.round(num);
console.log(roundedNum); // Output: 4

2. random

The Math.random() function returns a random number between 0 and 1:

var randomNum = Math.random();
console.log(randomNum); // Output: a random number between 0 and 1
